📥 Installation Methods

Method 1: Automatic Installation (Linux)

chmod +x install.sh sudo bash install.sh

Method 2: Manual Installation

  • 1. Install PHP 8.1, MySQL, Nginx
  • 2. Create database: CREATE DATABASE medical_lab;
  • 3. Copy application files
  • 4. Configure database credentials
  • 5. Set up SSL certificate
  • 6. Restart services

Method 3: Docker Container

docker pull medical-lab-reports docker run -p 80:80 medical-lab-reports

📋 System Requirements

Server

  • OS: Linux (Ubuntu 20.04 or higher)
  • Web Server: Nginx or Apache
  • PHP: 8.0 or higher
  • Database: MySQL 5.7 or higher
  • RAM: 2GB minimum
  • Disk: 10GB minimum
  • SSL: HTTPS certificate (Let's Encrypt)

Client (Web)

  • Browser: Chrome, Firefox, Safari, or Edge (latest)
  • Internet: Stable connection
  • JavaScript: Enabled

Client (Mobile)

  • Device: Android phone or tablet
  • OS: Android 7.0 (API 24) or higher
  • Storage: 200MB free
  • Internet: WiFi or mobile data
