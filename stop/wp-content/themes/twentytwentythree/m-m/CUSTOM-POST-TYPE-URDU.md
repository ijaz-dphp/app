# Custom Post Type - فوری گائیڈ (اردو)

## 🎉 کیا نیا شامل ہوا؟

آپ کی WordPress theme میں اب **Jobs** کے لیے ایک professional **Custom Post Type** شامل ہو گیا ہے!

## ✨ نئی Features

### 1. Jobs Custom Post Type
- WordPress admin میں الگ **Jobs** section
- Professional meta boxes
- Custom taxonomies (Categories اور Types)
- Beautiful job detail pages

### 2. Custom Meta Boxes (8 Fields)

#### Job Details:
- **Location** - جگہ (e.g., Dartford)
- **Salary** - تنخواہ (e.g., £26,000 – £32,000)
- **Contract Type** - نوکری کی قسم (Permanent, Contract, etc.)
- **Experience** - تجربہ (e.g., 2-5 years)
- **Qualifications** - قابلیت
- **Responsibilities** - ذمہ داریاں
- **Requirements** - ضروریات

#### Application Info:
- **Deadline** - آخری تاریخ
- **Contact Email** - رابطہ ای میل
- **Contact Phone** - رابطہ فون
- **Featured** - نمایاں کریں

### 3. نئی Template Files

1. **single-job.php** - ہر job کی تفصیل کا صفحہ
2. **archive-job.php** - تمام jobs کی لسٹ

## 📝 Job کیسے شامل کریں؟

### Step 1: WordPress Admin میں جائیں
**Jobs > Add New** پر کلک کریں

### Step 2: بنیادی معلومات
- **Title**: Job کا نام لکھیں (e.g., "Mechanical Supervisor")
- **Description**: تفصیل لکھیں

### Step 3: Job Details بھریں
نیچے scroll کر کے "Job Details" meta box میں:

```
Location: Dartford
Salary: £35,000 - £45,000
Contract Type: Permanent (dropdown سے منتخب کریں)
Experience: 3-5 years
```

**Qualifications** (ہر line پر ایک):
```
City & Guilds qualified
18th Edition certified
Valid CSCS card
```

**Responsibilities** (ہر line پر ایک):
```
Supervise M&E installation teams
Liaise with project managers
Ensure health and safety compliance
```

**Requirements** (ہر line پر ایک):
```
Minimum 3 years M&E experience
Strong leadership skills
Excellent communication
```

### Step 4: Application Information
دائیں جانب "Application Information" میں:
- **Deadline**: تاریخ منتخب کریں
- **Email**: applications@example.com
- **Phone**: +44 208 298 9977
- **Featured**: ✓ (اگر نمایاں کرنا ہو)

### Step 5: Categories اور Types
- **Job Categories** منتخب کریں (e.g., Mechanical, Electrical)
- **Job Types** شامل کریں (e.g., Permanent, Contract)

### Step 6: Publish کریں
**Publish** button پر کلک کریں!

## 🚀 Installation

1. پوری `m-m` folder کو ZIP بنائیں
2. WordPress admin: **Appearance > Themes > Add New > Upload**
3. ZIP upload کریں
4. **Activate** کریں
5. **Jobs > Add New** سے jobs شامل کرنا شروع کریں!
